M800AudioController

BT behavior

  1. onCallSessionCreated / onCallSessionDestroyed As name, should be called with call session created / destroyed, Should not start SCO mode and keep AudioManager mode in ringtone

    Only listen BT broadcast receiver event in call session active time

  2. onMediaEngineReady Call when in talking state, will start SCO mode and change AudioManager mode to in_communication

  3. Auto enable SCO when BT headset connected. Only check after media engine ready.
